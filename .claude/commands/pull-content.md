Help me pull content from BitCompass into this project.

## Steps

1. **Ask what I need.** Ask me what I'm looking for — a topic, keyword, or specific ID.

2. **Search.** Use `search-rules` (or `search-solutions` for solutions) to find matches. Present the results with ID, title, kind, and a short snippet.

3. **Select.** Ask me which item(s) to pull. I can select one or more.

4. **Pull.** For each selected item, call the `pull-rule` MCP tool with its ID. Report the file path written for each item.

5. **Summary.** List what was pulled and where the files are located.
